## 🚀 三步部署法

### 第一步：上传代码到 GitHub（5分钟）

1. **访问 GitHub：** https://github.com/new
2. **创建新仓库：**
   - 名称：`smart-teleprompter`
   - 类型：Public
   - 不要添加 README
3. **运行部署脚本：**
   ```bash
   cd /home/jack/ZRJ
   ./quick-deploy.sh
   ```
4. **按照提示上传代码**

---

### 第二步：部署后端到 Railway（10分钟）

1. **访问：** https://railway.app
2. **用 GitHub 登录**
3. **点击 "New Project"**
4. **选择 "Deploy from GitHub repo"**
5. **选择 `smart-teleprompter` 仓库**
6. **配置：**
   - Root Directory: `backend`
   - 添加环境变量：
     ```
     GOOGLE_API_KEY=38W44WK0ytmzBmRr1yArDwCypeY_3TewPiCkT9kcsQYhjKta7
     ```
7. **等待部署完成**（约2-3分钟）
8. **复制后端地址**（类似：`https://xxx.railway.app`）

---

### 第三步：部署前端到 Vercel（10分钟）

1. **访问：** https://vercel.com
2. **用 GitHub 登录**
3. **点击 "New Project"**
4. **选择 `smart-teleprompter` 仓库**
5. **配置：**
   - Framework Preset: `Create React App`
   - Root Directory: `frontend`
   - 添加环境变量：
     ```
     REACT_APP_BACKEND_URL=[刚才复制的Railway后端地址]
     ```
6. **点击 "Deploy"**
7. **等待部署完成**（约2-3分钟）
8. **获得前端地址**（类似：`https://smart-teleprompter.vercel.app`）
